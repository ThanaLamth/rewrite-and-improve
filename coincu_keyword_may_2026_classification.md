## Editorial Rule

Default rule for Coincu:

- publish from `keep`
- test carefully from `selective`
- reject from `drop`

If a keyword cannot be mapped back to:

- crypto education
- blockchain utility
- Bitcoin / Ethereum demand
- exchange / wallet / market research
- Bitcoin purchasing power / real-world asset comparison
- a clearly crypto-linked public figure

then it should not be in the main Coincu roadmap.
